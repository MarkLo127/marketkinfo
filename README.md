# MarketInfo

這是一個用於股市資訊分析應用程式，透過簡單搜索與簡潔頁面讓用戶快速取得資訊


## 目錄

- [功能](#功能)
- [安裝](#安裝)


## 功能

- 1.大盤指數
- 2.公司基本資訊
- 3.公司經營狀況
- 4.公司財報
- 5.交易數據
- 6.期權數據
- 7.SEC文件
- 8.機構買賣
- 9.近期相關消息

## 安裝

### 1. 克隆專案

```bash
git clone https://github.com/MarkLo127/marketkinfo.git
```

### 2. 進入專案目錄
```bash
cd marketkinfo
```
### 3. 安裝所需的依賴
```bash
pip install -r requirements.txt
```
### 4. 啟動應用程式
```bash
streamlit run main.py
```
